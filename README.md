# 项目启动
```
step1:
git clone https://github.com/xiebingjun777/start_project.git

step2:
cd start_project

step3:
npm install

step4:
npm run dev
```

# 技术栈
- vue-cli：Vue的官方脚手架工具
- Axios: 异步获取数据
- vue-router: Vue的路由管理插件
- ES6语法：更加简洁高效
- scss: 便捷，规范的css预处理器
- flex布局：自适应布局
- webpack: 模块压缩和项目打包器
